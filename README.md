# 🩺 Monitor Multiparâmetros - Simulador Clínico

Este projeto nasceu de uma necessidade real da Profa. Samantha (SENAC) e foi desenvolvido ao vivo/em bastidores pelo **Prof. Dr. Gerson Yoshinari** para o canal **Sandeco**.

O objetivo é fornecer um simulador de sinais vitais de alta fidelidade, totalmente gratuito e que funcione sem internet em laboratórios de prática profissional.

## 🚀 Funcionalidades
- **Ondas em Tempo Real:** ECG, Pletismografia (SpO2), Pressão Arterial e Capnografia.
- **Cálculos Automáticos:** Cálculo de PAM instantâneo.
- **Painel do Instrutor:** Edite valores, cores e nomes dos parâmetros sem recarregar a página.
- **Offline First:** Um único arquivo HTML. Baixou, rodou.
- **Vibecoding:** Desenvolvido através de iterações com IA, focando na intenção pedagógica.

## 🛠 Como usar
1. Baixe o arquivo `monitor.html`.
2. Abra em qualquer navegador (Chrome, Edge, Firefox).
3. Use o botão **⚙ Editar** para configurar o cenário clínico.
